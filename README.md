# Codex Usage Widget

一个轻量的 macOS 菜单栏工具，用于查看 Codex 五小时与周额度的剩余百分比、重置倒计时，以及赠送重置额度的到期提醒。

## 功能

- 根据 Codex 状态自动调整同步频率：任务活动 30 秒、前台空闲 1 分钟、后台空闲 10 分钟、客户端关闭 5 小时。
- 每次打开菜单也会立即刷新。
- 显示五小时与周额度的剩余百分比及重置倒计时；如果服务暂未提供五小时窗口，则自动只展示周额度，并用周额度更新菜单栏标题。
- 有可用赠送重置时，显示最早到期日期和“使用重置”按钮；没有可用次数时隐藏该模块。
- 使用重置前必须二次确认；不会自动消耗赠送次数。

## 使用

```sh
./build.sh
open "dist/Codex Usage.app"
```

## 隐私与安全

- 仅从本机 `~/.codex/auth.json` 读取当前 Codex 登录状态；不会记录、显示或上传凭据到第三方。
- 默认只查询额度与赠送重置状态。
- 只有用户点击“使用重置”并在确认框中再次确认后，才会向 Codex 提交一次重置请求；工具永远不会自动消耗赠送次数。
- 公开仓库不包含编译产物、缓存或本机认证文件。

## 注意

本工具依赖 Codex 当前客户端使用的账户接口，而非面向第三方长期承诺的公开 API。接口字段、访问规则或可用性发生变化时，工具可能需要更新。

“任务活动”通过本机 Codex 会话文件在最近 90 秒内持续写入来推断，因此是状态信号而不是官方任务状态 API。

需要 macOS 13 或更高版本，并已在 Codex 登录。构建只使用系统自带的 Cocoa 工具链，无需安装完整 Xcode。
